## HTML Features

The obfuscator can parse HTML files and identify `<script>` tags to obfuscate automatically.

```html
<!DOCTYPE html>
<html>
   <head>
      <script>
         // This code will be obfuscated
         console.log("Hello World");
      </script>
   </head>
   <body></body>
</html>
```
